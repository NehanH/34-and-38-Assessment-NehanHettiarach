Modelling

What is Blender Modelling?

Modelling in Blender is the process of creating 3D objects and shapes by manipulating vertices, edges, and faces in a 3D space. It's a fundamental skill in 3D design and is used in various fields such as animation, game design, and architecture.

Task 1

Basic Modelling: Create a simple 3D object like a cube or sphere and manipulate its shape.

1. Open Blender and create a new project.

2. Add a basic shape like a cube or sphere.

3. Use the move, rotate, and scale tools to manipulate the object's shape.

Task 2

Modelling With Modifiers: Use modifiers to automatically change an object's geometry.

1. Select an object in your scene.

2. Go to the Modifiers tab and add a modifier like Subdivision Surface.

3. Adjust the settings to see how the modifier affects the object.

Task 3

Extruding: Learn how to extrude faces to create complex shapes

1. Select a face on your object.

2. Press E to extrude the face and move it to create a new shape.

3. Experiment with extruding different faces to create complex objects.

Task 4

Sculpting In Modelling: Use sculpting techniques in the modelling process.

1. Switch to Sculpt Mode.

2. Use various brushes to shape your object.

3. Switch back to Object Mode to see the final model.

Task 5

Uv Mapping: Learn how to unwrap a 3D model and apply a texture using UV mapping.

1. Select your object and switch to Edit Mode.

2. Select all faces and press U to unwrap the object.

3. Apply a texture in the UV/Image Editor to see how it maps to the object.
